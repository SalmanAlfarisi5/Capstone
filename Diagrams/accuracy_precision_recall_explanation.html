Performance Metrics
Accuracy
Accuracy = (TP + TN) / (TP + TN + FP + FN)
Accuracy = (3 + 3) / (3 + 3 + 2 + 2) = 6/10 = 0.6 = 60%
What it measures: Overall correctness. 6 out of 10 predictions were correct = 60%
Precision
Precision = TP / (TP + FP)
Precision = 3 / (3 + 2) = 3/5 = 0.6 = 60%
What it measures: When predicting substitutable, how often is it correct? 3 out of 5 predictions were right = 60%
Recall
Recall = TP / (TP + FN)
Recall = 3 / (3 + 2) = 3/5 = 0.6 = 60%
What it measures: Of all substitutable pairs, how many did we find? Found 3 out of 5 = 60%
